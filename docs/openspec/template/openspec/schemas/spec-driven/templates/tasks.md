# Implementation Tasks

## Phase 1: Red (Write Tests)

- Write failing tests
- **Verifiable by:** Test fails

## Phase 2: Green (Make Tests Pass)

- Implement minimal code
- **Verifiable by:** Test passes

## Phase 3: Refactor (Improve Code)

- Improve code quality
- **Verifiable by:** Tests still pass, code improved

## Verification Checklist

- [ ] All tests pass (PACKAGE_MANAGER test:unit)
- [ ] Lint passes (PACKAGE_MANAGER lint)
- [ ] Type check passes (PACKAGE_MANAGER type-check)
- [ ] Format check passes (PACKAGE_MANAGER format-check)

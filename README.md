# Niuma (牛马)

> 一个智能生活助手，不只是工具，更是懂你的伙伴。

## 简介

Niuma 是一个处于早期开发阶段的智能生活助手项目。核心理念是打造一个不仅具备工具属性，更能提供"陪伴感"和"情绪价值"的智能精灵。

## 核心功能

### 🚀 开发效率
- 快速脚手架生成
- 任务拆解与进度追踪
- 结对编程辅助
- 错误分析与解决建议

### 📚 知识管理（Obsidian 联动）
- 笔记整理与关联发现
- 学习路径规划
- 复习提醒
- 知识问答

### 🎯 目标追踪
- 目标明确化
- 任务分解
- 进度可视化
- 敦促提醒

## 日循环陪伴

| 时段 | 行为 |
|------|------|
| 🌅 早晨 | 早安问候 + 天气 + 新闻 + 待办 |
| 🌙 晚间 | 一日总结 + 计划制定 |
| ☕ 空闲 | 灵感语录 / 小知识分享 |

## 技术栈

- **语言**: TypeScript / JavaScript
- **AI 接口**: iflow SDK
- **笔记联动**: Obsidian

## 开发状态

🚧 早期规划阶段
